# university_admin_workload_analytics
Analytics dashboard for administrators—requests, efficiency metrics, anomaly flags, and CSV/PDF exports.Built with Next.js, Tailwind, Recharts, and SQLite (CSV-seeded).  Includes My Tasks, Analytics, Reports, and Settings views with department/date filters.  Synthetic data generated via Python (Faker/NumPy/Pandas) for 2024–2025.
